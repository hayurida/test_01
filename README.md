# 🐍 Snake Game (HTML · CSS · JavaScript · Single File)

간단한 **스네이크 게임(Snake Game)** 을 하나의 HTML 파일 안에 완전히 구현한 프로젝트입니다.  
추가 설치 없이 브라우저에서 바로 실행할 수 있으며, **키보드 화살표**로 움직입니다.

---

## 📌 특징 (Features)

- 단일 HTML 파일(HTML + CSS + JS 포함)
- 키보드 화살표(← ↑ → ↓) 이동
- **P** → 일시정지 / 재개  
- **R** → 게임 재시작  
- 점수를 먹을수록 속도 점진 증가
- 캔버스 기반 렌더링
- 완전 오프라인 실행 가능

---

## 🎮 게임 방법 (How to Play)

| 키 | 기능 |
|----|------|
| ← ↑ → ↓ | 이동 |
| **P** | 일시정지 / 재개 |
| **R** | 게임 재시작 |

음식(빨간 블록)을 먹어 길이를 늘리고 가능한 오래 살아남는 것이 목표입니다.

---
