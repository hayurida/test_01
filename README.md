🐍 Snake Game (HTML · CSS · JavaScript · Single File)
간단하고 직관적인 스네이크 게임(Snake Game) 을 하나의 HTML 파일만으로 구현한 프로젝트입니다.
추가 파일 없이 브라우저만 있으면 실행할 수 있으며, 키보드 화살표로 조작합니다.

📌 특징(Features)

한 개의 HTML 파일에 HTML · CSS · JavaScript가 모두 포함
**키보드 화살표(← ↑ → ↓)**로 스네이크 움직임 조작
P 키 → 일시정지 / 재시작
R 키 → 게임 즉시 재시작
자연스러운 움직임 및 그리드 기반 렌더링
점수를 먹을수록 점점 빨라지는 난이도
깔끔한 UI, 반응형 크기 조정
설치, 빌드 과정 없이 바로 실행 가능


🎮 게임 방법(How to Play)





















키기능← ↑ → ↓이동P일시정지 / 재개R게임 재시작
목표는 음식(빨간 블록)을 먹어 스네이크 길이를 늘리고 최대한 오래 살아남는 것입니다.

📂 실행 방법(Run)

이 저장소를 다운로드하거나 클론합니다.
snake.html 파일을 더블 클릭하여 브라우저(Edge/Chrome 등) 에서 실행합니다.
바로 플레이 가능합니다.

Shellgit clone https://github.com/사용자명/저장소명.git더 많은 선 표시

📄 코드 구조(Code Structure)

모든 코드가 하나의 HTML 안에 들어 있습니다.

Plain Textsnake.html├── <style>  (CSS)└── <script> (JavaScript)``더 많은 선 표시
